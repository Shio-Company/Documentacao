# Backlog do Produto (User Stories)

## E-commerce (Core)

1. Como cliente, quero visualizar produtos disponíveis para poder escolher o que comprar
2. Como cliente, quero ver detalhes do produto (preço, descrição, imagens)
3. Como cliente, quero adicionar e remover produtos do carrinho
4. Como cliente, quero finalizar uma compra
5. Como cliente, quero escolher a forma de pagamento (cartão parcelado ou PIX)
6. Como cliente, quero visualizar o valor do frete antes de comprar
7. Como cliente, quero receber confirmação de compra

## Gestão de Produtos e Estoque

8. Como administrador, quero gerenciar produtos (cadastrar, editar e remover drops)
9. Como administrador, quero controlar estoque por produto
10. Como administrador, quero registrar preço de custo e venda
11. Como administrador, quero dar baixa automática no estoque após venda

## CRM e Dados

12. Como administrador, quero visualizar lista de clientes cadastrados
13. Como administrador, quero ver histórico de compras por cliente
14. Como administrador, quero analisar o comportamento dos clientes (frequência de compra, dados demográficos e recorrência)

## Marketing e Tracking

15. Como administrador, quero integrar pixel de rastreamento para acompanhar o comportamento dos usuários no site

## Painel Administrativo

16. Como administrador, quero acessar o sistema via mobile
17. Como administrador, quero gerenciar pedidos e acompanhar seus status
18. Como administrador, quero visualizar um dashboard com visão geral de vendas, estoque, pedidos e clientes

## Frete e Logística

19. Como sistema, quero calcular frete automaticamente
20. Como cliente, quero acompanhar status do pedido

## Plataforma

21. Como usuário, quero me cadastrar na plataforma
22. Como usuário, quero fazer login
23. Como usuário, quero recuperar senha

## Futuro (não MVP)

24. Como usuário, quero assinar um clube
25. Como usuário, quero acessar conteúdo exclusivo
26. Como usuário, quero usar cupom de influenciador
27. Como embaixador, quero receber comissão por indicação
