# ShioOficial — Guia de Estilo

> Identidade visual da Plataforma Shio: e-commerce gospel streetwear.

## Paleta de Cores

| Nome | Hex | Uso |
|------|-----|-----|
| Preto Shio | `#0D0D0D` | Fundo principal |
| Ouro | `#C9A84C` | Destaque, CTAs, links ativos |
| Ouro claro | `#E8C96A` | Hover, ênfase secundária |
| Branco Off | `#F0EDE6` | Texto principal |
| Cinza Médio | `#A0A0A0` | Texto secundário, legendas |
| Cinza Escuro | `#1A1A1A` | Fundo da sidebar |
| Cinza Borda | `#2A2A2A` | Bordas, divisores |

## Princípios de Design

- **Escassez e exclusividade:** visual limpo, sem ruído
- **Streetwear com propósito:** escuro, sofisticado, com toque dourado
- **Mobile-first:** todos os componentes devem funcionar em telas pequenas
- **Contraste alto:** acessibilidade é prioridade

---

## Componentes

### Botão primário
- Fundo: `#C9A84C`
- Texto: `#0D0D0D`
- Hover: `#E8C96A`
- Border-radius: `4px`

### Botão secundário
- Fundo: transparente
- Borda: `1px solid #C9A84C`
- Texto: `#C9A84C`
- Hover: fundo `#C9A84C`, texto `#0D0D0D`

### Links
- Cor padrão: `#C9A84C`
- Hover: `#E8C96A`
- Sem sublinhado por padrão

### Tabelas
- Cabeçalho: fundo `#1A1A1A`, texto `#C9A84C`
- Linhas alternadas: `#0D0D0D` e `#141414`
- Borda: `#2A2A2A`

### Código inline
- Fundo: `#1A1A1A`
- Texto: `#E8C96A`
- Border-radius: `3px`

### Bloco de código
- Fundo: `#111111`
- Borda esquerda: `3px solid #C9A84C`

### Blockquote
- Borda esquerda: `4px solid #C9A84C`
- Fundo: `#1A1A1A`
- Texto: `#A0A0A0`
