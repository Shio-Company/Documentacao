# Projeto Shio 

## 1. Objetivo do Projeto

Desenvolver uma plataforma digital para a marca Shio que integre e-commerce próprio, estrutura de comunidade (Club) e CRM. O foco é a escala do negócio e engajamento no nicho gospel.

## 2. Visão do Produto

A Shio passará a ser um ecossistema digital completo:

* **Roupas:** Venda de drops exclusivos.
* **Conteúdo:** Lançamento de livros e devocionais.
* **Comunidade:** Clube de membros com benefícios.

## 3. Escopo do MVP (0-3 meses)

O desenvolvimento inicial focará em funcionalidades essenciais:

* **E-commerce:** Gestão de drops, compras online e pagamentos via PIX/Cartão.
* **Gestão Interna:** Controle de estoque com preço de custo/venda e painel administrativo mobile.
* **CRM:** Cadastro de usuários e histórico de compras para análise de retenção.

## 4. Público-alvo

* **Perfil:** Jovens de 16 a 28 anos.
* **Nicho:** Consumidores de streetwear cristão/gospel com propósito.
